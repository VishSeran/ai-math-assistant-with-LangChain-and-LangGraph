# -ai-math-assistant-with-LangChain-and-LangGraph
Hands-on LangChain lab building AI agents with custom &amp; built-in tools. Covers Tool class vs @tool decorator, a multi-tool math toolkit (add/subtract/multiply/divide), debugging tool-agent mismatches, initialize_agent vs LangGraph's create_react_agent, Wikipedia tool integration, automated testing, and a power-tool exercise.
